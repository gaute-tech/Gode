# GODE AS Nettside

Enkel og responsiv nettside for GODE AS bygget med HTML og CSS.

## Om prosjektet

Dette er en statisk nettside hostet på GitHub Pages. Den er designet for å presentere GODE AS og våre tjenester.

## Innhold

- `index.html` - Hovedfil
- `styles.css` - Stilark
- `README.md` - Denne filen

## Hvordan kjøre lokalt

1. Last ned eller klon repositoryet
2. Åpne `index.html` i din nettleser

## Publisering på GitHub Pages

1. Gå til repository innstillinger
2. Scroll ned til "GitHub Pages"
3. Velg `main` som branch
4. Klikk "Save"

Din nettside vil da være tilgjengelig på: `https://gaute-techHei.github.io/Gode/`

## Personalisering

Du kan enkelt redigere:
- Firmainfo i `index.html`
- Farger og styling i `styles.css`
- Kontaktinformasjon og tjenester
